# 🌐 Network Packet Routing Simulator

A Python-based simulation of network routing using graph algorithms such as Dijkstra to compute the shortest path between nodes, representing packet transfer in real-world networks.

---

## 🚀 Features

- Graph-based network representation  
- Shortest path computation using Dijkstra’s Algorithm  
- Simulates packet routing between nodes  
- Simple CLI interface  

---

## 🧠 Concepts Used

- Graph (Adjacency List)
- Dijkstra Algorithm (Greedy + Priority Queue)
- Heap (Min Priority Queue)

---
